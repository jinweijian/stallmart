# 店铺装修静态资源

本目录存放小程序店铺装修风格包素材。路径由服务端 `StoreDecorationDTO` 和小程序 `STORE_THEME_PACKAGES` 共同约定。

当前 `forest/` 是“森系水果茶-小白款”第一版素材目录。后续替换 icon、banner、头图和吉祥物时保持同名路径，避免改动小程序页面代码。

素材落点：

- `home-banner.png` -> `forest/hero-forest-tea.jpg`（展示版）和 `forest/hero-forest-tea.png`（原始素材）
- `lunbo-1.png` -> `forest/banner-seasonal.jpg`（展示版）和 `forest/banner-seasonal.png`（原始素材）
- `lunbo-2.png` -> `forest/banner-tea.jpg`（展示版）和 `forest/banner-tea.png`（原始素材）
- `购物车.png` -> `forest/icons/cart.png`
- `类目icon1.png` -> `forest/icons/category-1.png`
- `类目icon2.png` -> `forest/icons/category-2.png`
