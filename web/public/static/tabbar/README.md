# Tabbar Icons

请在此目录放置 tabbar 图标文件：

- `home.png` - 首页（未选中）
- `home-active.png` - 首页（选中）
- `cart.png` - 购物车（未选中）
- `cart-active.png` - 购物车（选中）
- `order.png` - 订单（未选中）
- `order-active.png` - 订单（选中）
- `my.png` - 我的（未选中）
- `my-active.png` - 我的（选中）

推荐使用透明 PNG，并确保小程序原生 tabBar 引用的每个图标小于 40KB。处理时应先裁掉透明留白，让可见主体占画布 75% 到 90%，再在大小限制内保留尽可能高的分辨率。
